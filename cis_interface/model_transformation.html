Model Transformation¶

Model transformation refers to the process of modifying model code so that it can be integrated with other models using cis_interface. This sections covers the yaml entries for models in the different langauges as well as the basic API calls for sending/receiving string messages. If you are interested in the API calls for sending/receiving other types of messages, please see Formatted I/O.
